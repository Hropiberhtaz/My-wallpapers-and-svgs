## 🖼️ What's in here?

A curated collection of **white SVGs** and **wallpapers** that I think look pretty cool.

> ⚠️ **Disclaimer:** Don't judge the file names. I know they're kinda dumb. Just vibe with the visuals.

Some of these wallpapers are also available over at **[SleepyCatHey's repo](https://github.com/SleepyCatHey)** — I was just too lazy to go digging through there every time, so I made my own little stash. No regrets.
